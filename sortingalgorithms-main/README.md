# Sorting Algorithms Project

## 📌 Description
This is a menu-driven Python project that implements and compares 5 sorting algorithms:
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## 🚀 Features
- User input support
- Ascending / Descending sorting
- Modular code structure
- Easy-to-understand menu system

## 🛠 Technologies Used
- Python 3

## ▶ How to Run
```bash
python main.py
